# Intro to RUST Part (1/2) and (2/2)

## KVStore: Program that stores in a plain file "kv.db" key and values passed as arguments when it is executed.



### Commands employed

- cargo new kvstore
- cd kvstore
- tree
- cargo check
- cargo build
- cargo run
- ls target/debug
- cat README.md
- cargo build --release
- cargo run --release
- code .
- cargo run -- hello world
- xxd kv.db
- cargo run -- key value
